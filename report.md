- Configure the docker image configuration file (DockerFile)

      FROM python:3.7
      WORKDIR /app
      COPY requirements.txt .
      RUN pip install -r requirements.txt
      COPY . .
      CMD ["python", "main.py"]

- Build the docker image with the following command:

      sudo docker build -t souhair/simulator .
